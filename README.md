# AIspire Studio – AI-Powered Digital Branding Agency

**Overview:** AIspire Studio is a digital branding agency using AI to create logos, social media content, marketing copy, and pitch decks for startups.

**Phase 1 – Branding:** Created the business name, tagline, brand story, and AI-generated logo.

**Phase 2 – Marketing Content:** Generated social media posts, website mockups, and marketing copy using AI tools.

**Phase 3 – Client Outreach:** Developed sales pitches, outreach emails, and a 7-slide pitch deck to attract clients.

**Tools Used:** ChatGPT for text, Canva Magic Media for visuals.
